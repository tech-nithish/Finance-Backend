# 💰 Finance Dashboard Backend

A production-grade REST API backend for a finance dashboard system, built with **Spring Boot**, **Spring Security + JWT**, and **MongoDB Atlas**.

---

## 🧱 Tech Stack

| Layer | Technology |
|---|---|
| Framework | Spring Boot |
| Security | Spring Security + JWT |
| Database | MongoDB Atlas |
| ODM | Spring Data MongoDB |
| Validation | Jakarta Bean Validation |
| Docs | Swagger (OpenAPI) |
| Build | Maven |
| Language | Java |

---

## 🚀 Getting Started

### Prerequisites
- Java 17+
- Maven
- MongoDB Atlas connection

### Run the Application
```bash
./mvnw spring-boot:run
